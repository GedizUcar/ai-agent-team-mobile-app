# Terimler Sozlugu - Glossary

Bu belge, projede kullanilan Turkce ve Ingilizce teknik terimlerin karsilik tablosunu icerir. Tum dokumantasyonda tutarlilik saglamak icin bu sozluk referans alinir.

## Genel Yazilim Terimleri

| Turkce | Ingilizce | Aciklama |
|--------|-----------|----------|
| Gelistirme | Development | Yazilim gelistirme sureci |
| Gereksinim | Requirement | Proje ihtiyaclari |
| Mimari | Architecture | Yazilim mimarisi |
| Tasarim | Design | UI/UX ve sistem tasarimi |
| Uygulama | Implementation | Kodlama/gerceklestirme |
| Derleme | Build | Kod derleme |
| Dagitim | Deployment | Uygulamayi yayinlama |
| Surum | Release / Version | Yazilim versiyonu |
| Bakim | Maintenance | Yazilim bakimi |
| Hata | Bug | Yazilim hatasi |
| Duzeltme | Fix / Patch | Hata duzeltme |
| Yeniden duzenleme | Refactoring | Kod iyilestirme |
